# dump_ocamlformat

`dump_ocamlformat` dumps preset configuration of [`ocamlformat`](https://github.com/ocaml-ppx/ocamlformat).

## dependencies for dev

- `opam`

Install dependencies with the following command.

```bash
opam install -y . --deps-only
```
